# News Text Classification

本项目为《机器学习》课程实验二：新闻文本分类。实验任务是根据给定的新闻文本内容预测其所属类别标签。数据集中共有 200000 条新闻样本，类别数为 14 类，文本已经表示为由词 ID 组成的序列，是一个典型的自然语言处理多分类任务。

项目基于 PyTorch 实现了一个轻量级文本分类模型，模型结构为 `Embedding mean/max pooling + MLP`。首先将输入的 token ID 序列映射为词向量表示，然后分别进行平均池化和最大池化，以提取文本整体主题信息和局部关键词特征；随后将两种池化结果拼接，并输入多层感知机完成 14 类新闻文本分类。

## 项目内容

* 新闻文本数据读取与探索分析
* 文本长度统计与类别分布可视化
* 文本序列截断与填充处理
* 训练集、验证集和测试集划分
* 基于 Embedding + Pooling + MLP 的文本分类模型
* Softmax 与交叉熵损失函数训练
* Accuracy、Macro-F1、ABS-SUM 等指标评价
* 分类报告与混淆矩阵分析

## 模型结构

本实验采用的模型结构如下：

1. 输入层：定长 token ID 序列，输入长度为 256；
2. 词嵌入层：将 token ID 映射为 128 维词向量；
3. 池化层：同时使用 Mean Pooling 和 Max Pooling；
4. 隐藏层：Linear + ReLU + Dropout；
5. 输出层：Linear 映射到 14 个类别；
6. 预测层：Softmax 输出类别概率。

## 实验结果

在 40000 条测试样本上，模型取得如下结果：

| 指标              |       结果 |
| --------------- | -------: |
| Accuracy        |   0.8976 |
| Macro Precision |   0.8720 |
| Macro Recall    |   0.8784 |
| Macro F1        |   0.8746 |
| Weighted F1     |   0.8978 |
| Mean ABS-SUM    | 0.327815 |

实验结果表明，该模型能够较好地完成新闻文本分类任务。类别 2、类别 3、类别 6 等类别识别效果较好，而类别 7、类别 10、类别 11 等类别仍存在一定混淆，说明不同新闻主题之间可能存在语义重叠。
